# api3-telegram-bot

There are three Api3 groups in Telegram. In Telegram a group is sometime called a chat, supergroup, or chat group.

- Api3 - public with invite URL
- Api3 Administrators - private
- Api3 Logging - private

This bot checks each Telegram message using an AI prompt. This determines if a message violates the Api3 group rules. The rules are publicly available, the AI prompt is not.

The bot also provides a new user welcome message with an "are you human challenge".
